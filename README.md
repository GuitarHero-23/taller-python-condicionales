Descripción: El repositorio contiene ejercicios propuestos en Programación en Python, con el principal objetivo de practicar el uso de Git y Git Hub, con la creación y el uso de repositorios propios.
Ejercicios:
1) Saludo Formateado: El programa pide insertar el nombre, la edad y la ciudad de recidencia para otorgar un saludo personalizado.
2) Mayoría de edad: El programa pide inicialmente insertar un número entero, a partir de ese dato se identifica si el número insertado fue par o impar. El programa también pide insertar su edad, a partir del número insertado se identifica si el usuario es mayor de edad o menor de edad.
3) Descuento Tienda: El programa pide insertar el subtotal de una compra y si es un cliente Vip o Regular, según los datos proporcionados se aplica un descuento si cumple las condiciones y se imprime un mensaje señalando el Subtotal, el Descuento Aplicado y el Total Final.
4) Tarifa de Taxi: El programa pide insertar los kilómetros recorridos en una carrera y la hora de inicio del viaje para otorgarle al usuario la factura total del viaje, aplicando montos segun si se cumplen las condiciones.
5) Clasificador de Notas: El programa pide insertar tres notas de un estudiante, según las cuales se sacará un promedio y se indicará si el estudiante reprueba, entra a suletorios, conigue buena nota, consigue muy buena nota o consigue una excelente nota.
